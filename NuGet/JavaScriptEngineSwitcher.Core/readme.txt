   ===========
   DESCRIPTION
   ===========   
   JavaScript Engine Switcher determines unified interface for access to 
   the basic features of popular JavaScript engines (MSIE JavaScript 
   Engine for .Net, Microsoft ClearScript.V8, Jurassic and Jint). This
   library allows you to quickly and easily switch to using of another
   JavaScript engine.
